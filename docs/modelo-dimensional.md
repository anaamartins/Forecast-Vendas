# Modelo dimensional

## Objetivo

O modelo dimensional foi desenvolvido para organizar a informação comercial e financeira e suportar as análises realizadas no Power BI.

## Granularidade

A granularidade das tabelas de factos e dimensões foi definida de acordo com os processos de negócio e as necessidades de análise.

## Tabelas de factos

Nesta secção são apresentadas as tabelas de factos, as respetivas medidas e a granularidade dos registos.

## Tabelas de dimensões

Nesta secção são apresentadas as dimensões utilizadas para analisar as vendas por cliente, mercado, país, empresa, linha de negócio e período.

## Relações

As tabelas de factos estão relacionadas com as dimensões através das respetivas chaves.
