# Modelo semântico

## Objetivo

O modelo semântico foi desenvolvido em SQL Server Analysis Services para disponibilizar uma camada de negócio centralizada e consistente.

## Fonte de dados

O modelo tabular consome os dados armazenados no Data Warehouse.

## Componentes

O modelo inclui:

- tabelas de factos e dimensões;
- relações;
- medidas;
- hierarquias;
- roles;
- partições.

## Power BI

O Power BI consome o modelo tabular através de uma ligação live.
